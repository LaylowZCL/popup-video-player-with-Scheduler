# Popup Video Player with Scheduler

Aplicação desktop multiplataforma que exibe vídeos em popups em horários agendados.

## 📦 Funcionalidades
- Agendamento inteligente de exibições
- Player de vídeo em janela sempre no topo
- Integração com API de conteúdo
- Configuração intuitiva

## 🛠 Instalação
```bash
# Clone o repositório
git clone https://github.com/LaylowZCL/popup-video-player-with-scheduler.git

# Instale as dependências
npm install

# Execute em modo desenvolvimento
npm start
```

## 📦 Build para produção
```bash
# Windows
npm run dist:win

# macOS
npm run dist:mac
```
